# قائمة تدقيق نشر المشروع

## قائمة التحقق الأمنية

### الإعدادات الحساسة
- [ ] غيّر `SECRET_KEY` إلى مفتاح عشوائي قوي
  ```bash
  python -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'
  ```
- [ ] اضبط `DEBUG = False` في الإنتاج
- [ ] حدّث `ALLOWED_HOSTS` بدومينك الفعلي
- [ ] أنشئ ملف `.env` آمن بدلاً من hardcoding المعلومات

### أمان قاعدة البيانات
- [ ] انقل من SQLite إلى PostgreSQL للإنتاج
  ```bash
  pip install psycopg2-binary
  ```
- [ ] أنشئ مستخدم قاعدة بيانات منفصل
- [ ] استخدم كلمات مرور قوية
- [ ] فعّل النسخ الاحتياطية التلقائية

### أمان الملفات الثابتة والوسائط
- [ ] اجمع الملفات الثابتة: `python manage.py collectstatic`
- [ ] استخدم CDN لخدمة الملفات الثابتة
- [ ] خزّن الصور على S3 أو خدمة سحابية
- [ ] افصل مجلد الوسائط عن جذر المشروع

### أمان الاتصالات
- [ ] اشتر شهادة SSL/TLS
- [ ] اضبط `SECURE_SSL_REDIRECT = True`
- [ ] اضبط `SESSION_COOKIE_SECURE = True`
- [ ] اضبط `CSRF_COOKIE_SECURE = True`
- [ ] أضف رؤوس أمان إضافية

## إعدادات الأداء

- [ ] فعّل ذاكرة التخزين المؤقت (Redis/Memcached)
- [ ] استخدم CDN لخدمة الملفات الثابتة
- [ ] ضغط صور المنتجات
- [ ] قلل استعلامات قاعدة البيانات (ORM optimization)
- [ ] فعّل GZIP compression

## قاعدة البيانات والهجرات

- [ ] تشغيل `python manage.py migrate` على الخادم
- [ ] التحقق من الطراز اليدوي بعد الهجرة
- [ ] عملية الرجوع الآمن للبيانات

## اختبار ما قبل النشر

- [ ] تشغيل جميع الاختبارات
  ```bash
  python manage.py test
  ```
- [ ] فحص النظام
  ```bash
  python manage.py check --deploy
  ```
- [ ] اختبار محاكاة الإنتاج محلياً
  ```bash
  DEBUG=False python manage.py runserver
  ```

## الخادم والنشر

### إذا كنت تستخدم Heroku
```bash
heroku config:set SECRET_KEY='your-new-secret-key'
heroku config:set DEBUG=False
heroku config:set ALLOWED_HOSTS='yourdomain.com,www.yourdomain.com'
git push heroku main
heroku run python manage.py migrate
heroku run python manage.py createsuperuser
```

### إذا كنت تستخدم خادم Linux مخصص
- [ ] ثبّت Gunicorn: `pip install gunicorn`
- [ ] أنشئ ملف خدمة systemd لـ Gunicorn
- [ ] ثبّت و أعد تشكيل Nginx كـ reverse proxy
- [ ] فعّل Let's Encrypt لـ SSL مجاني

### إذا كنت تستخدم PythonAnywhere
- [ ] حمّل الملفات
- [ ] أنشئ بيئة افتراضية
- [ ] اضبط أوامر WSGI
- [ ] حدّث ملف إعدادات الويب
